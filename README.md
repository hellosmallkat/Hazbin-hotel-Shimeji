## 🐾 What is a Shimeji?

Shimeji are small animated desktop companions that roam freely around your screen. They walk on the edges of your windows, fall from the top of the screen, climb walls, and generally make your desktop a little more lively. You can drag them around with your mouse, and right-click them for options like spawning more or sending them home.

---

## 💻 Requirements

- **Windows 10 or 11** (Shimeji-ee is Windows only)
- **Java** — required to run Shimeji-ee ([Download Java here](https://www.java.com/en/download/))
- **Shimeji-ee** — the program that runs the shimeji ([Download from kilkakon.com](https://kilkakon.com/shimeji/shimejiee.zip))
- A way to unzip files — Windows has this built in, or use [7-Zip](https://www.7-zip.org/) (free)

## 🛠️ Installation

### Step 1 — Download & Install Java

1. Go to [java.com](https://www.java.com/en/download/)
2. Click **Free Java Download**
3. Run the installer and follow the prompts

### Step 2 — Download Shimeji-ee

1. Go to [kilkakon.com/shimeji](https://kilkakon.com/shimeji)
2. Click **Download Now** — this will download a file called `shimejiee.zip`
3. Right-click the zip file and select **Extract All...**
4. Choose a permanent location (like `C:\Shimeji\`) — **don't leave it in your Downloads folder!**

You should now have a folder that contains `Shimeji-ee.jar` and an `img` folder inside it.

### Step 3 — Download a Shimeji from This Repo

1. On this GitHub page, click the folder for the character you want
2. Click the green **Code** button → **Download ZIP**
   - Or download individual character folders if you only want specific ones
3. Extract the downloaded zip file

## 🎨 Adding a Character

Each character in this repo is its own folder. To add one to Shimeji-ee:

1. Open your **Shimeji-ee folder** (wherever you extracted it in Step 2)
2. Open the **`img`** subfolder inside it
3. Copy the character's folder (e.g. `Alastor`) into the `img` folder
   - The final path should look like: `Shimeji-ee/img/Alastor/`
4. Make sure the character folder contains both the image files **and** a `conf` folder

That's it! The character is now ready to use.

---

## ▶️ Running Your Shimeji

1. Open your **Shimeji-ee folder**
2. Double-click **`Shimeji-ee.jar`** to launch the program
   - If nothing happens, make sure Java is installed correctly
   - If Windows warns you about an unknown publisher, click **More info → Run anyway**
3. A small icon will appear in your **system tray** (bottom-right corner of your taskbar)
4. Right-click the tray icon → **Choose Shimeji...** → select your character
5. Your shimeji will appear and start exploring your screen! 🎉

---

## 🖥️ Creating a Desktop Shortcut

So you don't have to dig through folders every time:

1. Navigate to your **Shimeji-ee folder**
2. Right-click **`Shimeji-ee.jar`**
3. Select **Send to → Desktop (create shortcut)**
4. Your shortcut will now appear on your desktop — just double-click it to launch!

> **Tip:** You can rename the shortcut to something like "My Shimeji" by right-clicking it and selecting **Rename**.

---

## 🖱️ Using Your Shimeji

Once your shimeji is running, here's what you can do:

| Action | How to do it |
|---|---|
| Grab & move them | Click and drag with your mouse |
| Spawn more | Right-click the shimeji or tray icon → **Call Another** |
| Send them all home | Right-click tray icon → **Dismiss All** |
| Switch character | Right-click tray icon → **Choose Shimeji...** |
| Close the app | Right-click tray icon → **Bye-bye!** |
